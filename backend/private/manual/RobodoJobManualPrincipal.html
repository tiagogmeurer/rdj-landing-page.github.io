Entenda cada elemento do prompt acima:



Definição da Identidade (parâmetro Identity):


  "identity": {

    "character_type": "adult female",

    "age_range": "early 20s",

    "ethnicity": "white",

    "vibe": "soft innocent beauty with subtle sensual undertones"

  },




identity -> character_type: define o tipo do personagem, no caso do exemplo, “adult female”, ou seja, fêmea adulta, que será nossa modelo robô do job.
identity -> age_range: define a faixa etária da modelo (idade).
identity -> ethnicity: define a etnia da modelo (coloque qualquer uma da sua preferência).
identity -> vibe: descreva aqui o sentimento que você quer que a modelo transmita.

Definição de Atributos Físicos (parâmetro physical_attributes): 


  "physical_attributes": {

    "body": {

      "build": "slim",

      "proportions": "delicate feminine curves",

      "posture": "relaxed posture with subtle chest-forward tilt"

    },

    "skin": {

      "tone": "fair",

      "texture": "natural realistic skin",

      "details": "light freckles across nose and cheeks"

    },

    "face": {

      "shape": "soft oval",

      "jawline": "delicate",

      "cheeks": "slightly flushed",

      "lips": "soft full lips, slightly parted"

    },

    "eyes": {

      "color": "green",

      "shape": "large expressive eyes",

      "expression": "calm seductive gaze, warm and inviting"

    },

    "hair": {

      "color": "natural red",

      "shade": "warm copper ginger",

      "style": "long hair framing the face and falling over shoulders",

      "texture": "realistic strands with gentle movement"

    }

  },


physical_attributes -> body -> build: define o tipo físico, ex: gorda, magra etc

physical_attributes -> body -> proportions: define as proporções do corpo de forma mais detalhada.
physical_attributes -> body -> posture: define a postura corporal da modelo.

physical_attributes -> skin -> tone: tonalidade da pele.

physical_attributes -> skin -> texture: textura da pele.
physical_attributes -> skin -> details: detalhes da pele.

physical_attributes -> face -> shape: formato do rosto.
physical_attributes -> face -> jawline: formato da linha do queixo.
physical_attributes -> face -> cheeks: tipo de bochecha.
physical_attributes -> face -> lips: tipo de lábios.
physical_attributes -> eyes -> color: cor dos olhos.
physical_attributes -> eyes -> shape: formato dos olhos.
physical_attributes -> eyes -> expression: expressão dos olhos.
physical_attributes -> hair -> color: coloração dos cabelos.
physical_attributes -> hair -> shade: sombreamento dos cabelos.
physical_attributes -> hair -> style: estilo do corte dos cabelos.
physical_attributes -> hair -> texture: define a textura dos cabelos.        

 
Definição das Expressões Faciais (parâmetro facial_expression):


  "facial_expression": {

    "emotion": "soft confidence",

    "mouth": "subtle relaxed smile with parted lips",

    "eyes_focus": "direct eye contact",

    "overall_feel": "sensual but approachable"

  },


facial_expression -> emotion: define a emoção que a face deve transmitir.

facial_expression -> mouth: expressão dos lábios.
facial_expression -> eyes_focus: define onde deve estar o foco do olhar da modelo.
facial_expression -> overall_feel: define o sentimento geral que a face da modelo deve transmitir.


Definição do Estilo (parâmetro styling):


    "styling": {

    "makeup": {

      "style": "natural sensual makeup",

      "details": "light mascara, subtle blush, soft lip gloss"

    },

    "clothing": {

      "description": "feminine outfit showing off a daring neckline",

      "neckline": "soft V-neck or scoop neckline showing natural cleavage",

      "fabric": "light fabric gently contouring the upper body",

      "tone": "neutral or warm soft colors",

      "exposure_level": "suggestive but non-explicit"

    }

  },

styling -> makeup -> style: define o estilo da maquiagem.

styling -> makeup -> details: detalhamento da maquiagem.  

styling -> clothing -> description: descrição da roupagem.
styling -> clothing -> neckline: tipo da gola da camisa.
styling -> clothing -> fabric: tipo do tecido da roupa.
styling -> clothing -> tone: tonalidade de cor da roupa.
styling -> clothing -> exposure_level: nível de exposição do corpo.


Definição do Estilo Visual (parâmetro visual_style):


  "visual_style": {

    "realism_level": "ultra realistic",

    "lighting": "soft diffused lighting emphasizing skin and collarbone",

    "color_grading": "warm cinematic tones",

    "skin_detail": "high realism with subtle natural imperfections"

  },


visual_style -> realism_level: define o nível de realismo da imagem. (use ultra realistic para melhores resultados).

visual_style -> lighting: definição do estilo de iluminação da fotografia.
visual_style -> color_grading: define o gradiente da coloração da fotografia.
visual_style -> skin_detail: define o detalhamento geral da pele da modelo.





Definições de Câmera (parâmetro camera):

  "camera": {

    "shot_type": "vertical portrait",

    "aspect_ratio": "9:16",

    "angle": "slightly below eye level to subtly emphasize neckline",

    "framing": "head, shoulders and upper chest",

    "depth_of_field": "shallow",

    "background": "brazilian low-income middle class female room"

  },

camera -> shot_type: define a orientação da fotografia.
camera -> aspect_ratio: define o aspecto da tela (9:16 para celulares, 16:9 para computador).

camera -> angle: define o ângulo da câmera.
camera -> framing: define quais partes do corpo da modelo deve aparecer no enquadramento.
camera -> depth_of_field: define a profundidade do fundo da imagem.
camera -> background: define o cenário do fundo da imagem.  



Definições Negativas de Prompt (parâmetro negative_prompt):

  "negative_prompt": [

    "cartoon",

    "anime",

    "cgi",

    "plastic skin",

    "harsh lighting",

    "distorted face"

  ],


negative_prompt: aqui, inclua em aspas depois do sinal [ critérios para serem bloqueados na criação da imagem, no caso do nosso exemplo:

“cartoon”: para evitar que a imagem seja gerada com estilo de cartoon.

“anime”: para evitar que a imagem seja gerada com estilo de anime.

“cgi”: evita imagem de CGI.

“plastic skin”: evita que a pele da modelo tenha aspecto plástico.

“harsh lighting”: evita iluminação muito forte.

“distorted face”: evita que o rosto da modelo saia distorcido.



Configurações de Geração da Imagem (parâmetro generation_settings):


  "generation_settings": {

    "quality": "high",

    "detail_boost": true,

    "consistency_priority": true,

    "mobile_optimized": true

  }

generation_settings -> quality: defina aqui a qualidade da imagem (low, medium ou high).
generation_settings -> detail_boost: para dar um boost no detalhamento geral da imagem coloque true, se não quiser coloque false.
generation_settings -> consistency_priority: esse é um importante parâmetro, ele ajuda a manter os aspectos originais do modelo, por mais que você mude o cenário ou posição dela nas imagens, as características são mantidas. Deixe true para manter a consistência das características, caso não se importe em manter, coloque false.
generation_settings -> mobile_optimized: true para otimizar a imagem para o celular, false se for indiferente.
